## Rules
- Eating food makes the snake grow.
- When food is eaten it moves to another random place.
- Snake wraps around the screen (optional).
- Game over when snake hits itself (or the wall).

## Settings
- wrap snake
- speed of game
- shape of game (sharp/rounded)
- theme
    - default
    - rose-pine
    - catppuccin mocha
    - gruvbox dark
    - solarized dark
    - tokyo night
    - cyberpunk
    - one dark
    - nord
    - kangawa
- enable sound (eat food/ snake die)

## Controls
1. Arrow keys
2. wasd
3. hjkl
